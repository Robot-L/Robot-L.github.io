如何快速搭建Hexo博客

发表于 2019-12-07 分类于 胡搞瞎搞 ， 博客 阅读次数： Valine：

说来惭愧，16年我第一次使用Hexo搭建的github博客（就是眼前这个博客），写了第一篇博文【序】，say了一声hello world，就直接打入冷宫，3年过去还是那一篇博文…最近想把平时写的文章重新整理放到博客上，突然发现Hexo早就不会用了，赶紧重新找资料温习了一番，也趁此次机会记录一下搭建的全过程，或许能帮助需要的朋友。废话不多说，来看看如何建立一个功能强大，颜值还高的静态博客。

搭建最简单的Hexo博客

0.首先需要安装git和node.js，直接网上搜索下载安装包进行安装。

装完node.js就能使用npm命令（Node Package Manager）了，npm可以理解成一个前端的maven，用来管理js包，以后需要安装前端的东西都可以使用npm。

1.安装npm后，用npm命令安装hexo

2.初始化博客项目

4.启动本地服务

等待一会，访问http://localhost:4000就能看到刚生成的博客网站了，里面有一篇hello world的文章是hexo自己生成的。

5.写作

文章目录在source/_posts/，已经有一个hello-world.md的范文，我们把自己的文章文件（必须是md文件）拷贝到这个目录下即可，建议在每篇文章的开头部分添加以下信息，指定标题和创建时间，hexo需要这些信息在页面上进行展示。

如果你很懒，不写这段头部信息也是可以，不过会默认title为文件名，文章创建时间为文件的生成时间。

最后刷新网页即可看到新添加的文章。
